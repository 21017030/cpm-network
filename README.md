# CPM 네트워크

임계 경로법(Critical Path Method)을 시각화하는 웹 애플리케이션입니다.

작업 목록과 선행 관계를 입력하면 CPM 네트워크 다이어그램과 임계 경로를 자동으로 계산합니다.

## 주요 기능

- **작업 입력**: 작업명, 설명, 선행 작업, 소요 기간을 테이블 형태로 입력
- **시간 단위 선택**: 일(day) / 주(week) 단위 지원
- **예시 불러오기**: 쇼핑몰 개발, 소프트웨어 개발, 풀스택 서비스 개발 등 3가지 예시 프로젝트 제공
- **그래프 생성**: 입력된 작업 데이터를 기반으로 CPM 네트워크 다이어그램 자동 생성
- **임계 경로 시각화**: 임계 경로(TF = 0)를 빨간색으로 강조 표시
- **최소 완료 기간 계산**: 프로젝트의 최소 완료 기간을 자동으로 계산
- **노드 상세 보기**: 각 작업 노드를 클릭하거나 전체 상세 보기 모드로 ES, DR, EF, LS, TF, LF 값 확인
- **드래그로 노드 이동**: 그래프 내 노드를 자유롭게 드래그하여 레이아웃 조정
- **CPM 가이드 페이지**: CPM의 개념, 계산 절차, 예시를 설명하는 안내 페이지 제공

## CPM이란?

CPM(임계 경로법, Critical Path Method)은 프로젝트를 구성하는 작업들의 선행 관계를 네트워크 다이어그램으로 표현하고, 프로젝트 최소 완료 기간을 결정하는 가장 긴 경로를 찾아내는 일정 관리 기법입니다.

각 작업 노드는 다음 6가지 값을 표시합니다:

| 기호 | 영문 명칭 | 의미 |
|------|-----------|------|
| ES | Earliest Start | 최초 착수 가능 시점 |
| DR | Duration | 작업 소요 기간 |
| EF | Earliest Finish | 최초 완료 가능 시점 |
| LS | Latest Start | 최종 착수 가능 시점 |
| TF | Total Float | 총 여유 시간 |
| LF | Latest Finish | 최종 완료 가능 시점 |

## 기술 스택

- **React 18** + **TypeScript**
- **Vite** (빌드 도구)
- **@xyflow/react** (ReactFlow v12) — 네트워크 그래프 시각화
- **@dagrejs/dagre** — 그래프 레이아웃 계산
