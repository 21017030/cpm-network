# CPM 네트워크 생성기

임계 경로법(Critical Path Method)을 시각화하는 웹 애플리케이션입니다.  
작업 목록과 선행 관계를 입력하면 CPM 네트워크 다이어그램과 임계 경로를 자동으로 계산합니다.

---

## 주요 기능

### 작업 입력
- 작업명, 설명, 선행 작업, 소요 기간을 테이블 형태로 입력
- 시간 단위를 일(day) / 주(week) 중 선택 가능
- 항목 추가 / 삭제 / 전체 초기화 지원
- 쇼핑몰 개발, 소프트웨어 개발, 풀스택 서비스 개발 등 3가지 예시 프로젝트 즉시 불러오기

### 그래프 생성 및 시각화
- 입력 데이터를 기반으로 CPM 네트워크 다이어그램 자동 생성
- 임계 경로(TF = 0)는 빨간색 노드와 선으로 강조 표시
- 일반 경로는 파란색으로 구분
- 프로젝트 최소 완료 기간과 임계 경로를 그래프 상단에 요약 표시

### 노드 보기 모드
- **원형 모드 (기본)**: 작업명과 소요 기간만 간결하게 표시
- **개별 상세 보기**: 원형 노드 클릭 시 해당 노드만 ES/DR/EF/LS/TF/LF 6칸 박스로 전환
- **전체 상세 보기**: 하단 체크박스로 모든 노드를 한 번에 상세 보기로 전환
- 상세 모드에서 노드에 마우스를 올리면 작업 설명이 툴팁으로 표시

### 그래프 조작
- 노드를 드래그하여 레이아웃 자유롭게 조정
- 원형↔상세 모드 전환 시 드래그로 이동한 위치 유지 (노드 크기 차이만큼 비율 보정)
- 중앙으로 버튼으로 그래프를 화면에 맞게 재정렬
- 위치 초기화 버튼으로 자동 레이아웃으로 복원
- 도움말 버튼으로 노드 구조 설명 팝업 확인

### CPM 가이드 페이지
- 우측 상단 "CPM이란?" 링크로 이동
- CPM 개념 소개, 노드 구조 설명, 계산 절차(전진/후진/여유시간), 계산 예시, 활용법 수록
- 각 항목(ES/DR/EF/LS/TF/LF)의 상세 설명을 아코디언으로 펼쳐볼 수 있음

## CPM이란?

CPM(임계 경로법, Critical Path Method)은 프로젝트를 구성하는 작업들의 선행 관계를  
네트워크 다이어그램으로 표현하고, **프로젝트 최소 완료 기간을 결정하는 가장 긴 경로**를  
찾아내는 일정 관리 기법입니다.

### 계산 흐름

1. **전진 계산** (앞→뒤): ES, EF 계산
   - `EF = ES + DR`
   - 후속 작업의 `ES = max(선행 작업들의 EF)`

2. **후진 계산** (뒤→앞): LF, LS 계산
   - `LS = LF − DR`
   - 선행 작업의 `LF = min(후속 작업들의 LS)`

3. **여유 시간 계산**: `TF = LS − ES`
   - `TF = 0` → 임계 작업 (지연 시 전체 일정 지연)
   - `TF > 0` → 비임계 작업 (TF 범위 내에서 조정 가능)

### 노드 구조

각 작업은 6칸 박스로 표현됩니다:

```
┌──────┬──────┬──────┐
│  ES  │  DR  │  EF  │
├──────┴──────┴──────┤
│       작업명        │
├──────┬──────┬──────┤
│  LS  │  TF  │  LF  │
└──────┴──────┴──────┘
```

| 기호 | 영문 명칭 | 의미 |
|------|-----------|------|
| ES | Earliest Start  | 최초 착수 가능 시점 |
| DR | Duration        | 작업 소요 기간 |
| EF | Earliest Finish | 최초 완료 가능 시점 |
| LS | Latest Start    | 최종 착수 가능 시점 |
| TF | Total Float     | 총 여유 시간 |
| LF | Latest Finish   | 최종 완료 가능 시점 |

## 프로젝트 구조

```
src/
├── App.tsx               # 최상위 컴포넌트 — 전체 상태 관리 및 컴포넌트 조합
├── types.ts              # 공통 타입 정의 (Activity, Unit)
├── main.tsx              # React 앱 진입점
│
├── lib/                  # 비즈니스 로직 (UI와 무관한 순수 함수)
│   └── cpm.ts            # CPM 알고리즘 (위상 정렬, 전진/후진 계산, 임계 경로 판별)
│
├── hooks/                # 커스텀 React 훅
│   └── useCpmLayout.ts   # ReactFlow 노드 레이아웃 계산 및 위치 상태 관리
│
└── components/           # UI 컴포넌트
    ├── ActivityTable.tsx     # 작업 입력 테이블 + 예시 불러오기 드롭다운
    ├── PredecessorSelect.tsx # 선행 작업 다중 선택 드롭다운 (Portal 기반)
    ├── CpmNode.tsx           # CPM 노드 컴포넌트 (원형 / 상세 박스 두 가지 모드)
    ├── NetworkGraph.tsx      # ReactFlow 네트워크 그래프 + 컨트롤 바
    └── CpmGuide.tsx          # CPM 개념 가이드 페이지
```
