================================================================
CPM 네트워크 생성기 - 디렉토리 구조 설명
================================================================

📁 src/
│
├── 📄 types.ts
│     앱 전체에서 공유하는 타입 정의.
│     Activity(작업 한 행), Unit(일/주) 등 여러 파일에서 참조하는
│     공통 타입은 여기에 모아둔다.
│
├── 📄 App.tsx
│     최상위 컴포넌트. 전체 상태(작업 목록, 단위, 계산 결과)를 관리하고
│     하위 컴포넌트들을 조합한다.
│
├── 📄 main.tsx
│     React 앱의 진입점. App 컴포넌트를 DOM에 마운트한다.
│     일반적으로 직접 수정할 일이 없다.
│
│
├── 📁 lib/           ── 비즈니스 로직 (UI와 무관한 순수 계산)
│   │
│   └── 📄 cpm.ts
│         CPM 알고리즘 구현.
│         전진 계산(ES, EF), 후진 계산(LS, LF), 여유 시간(TF),
│         임계 경로 판별, 위상 정렬 등 핵심 로직이 모여 있다.
│         React에 의존하지 않는 순수 TypeScript 함수이므로
│         UI와 완전히 분리되어 있다.
│
│         → lib/ 에는 이처럼 UI 없이 독립적으로 동작하는 유틸리티,
│           알고리즘, API 통신 함수 등을 넣는다.
│
│
├── 📁 hooks/         ── 커스텀 React 훅
│   │
│   └── 📄 useCpmLayout.ts
│         ReactFlow 노드/엣지 레이아웃을 계산하고 상태를 관리하는 훅.
│         CPM 결과를 받아 화면 배치 좌표를 계산하고,
│         원형↔상세 모드 전환 시 위치 보정 로직을 담당한다.
│         NetworkGraph 컴포넌트에서 호출해 사용한다.
│
│         → hooks/ 에는 useState, useEffect 등 React 훅을 조합하여
│           만든 재사용 가능한 커스텀 훅을 넣는다.
│           컴포넌트처럼 UI를 렌더링하지는 않고,
│           상태와 로직만 다루는 함수들이 이 폴더에 속한다.
│
│
└── 📁 components/    ── UI 컴포넌트 (화면에 그려지는 요소들)
    │
    ├── 📄 ActivityTable.tsx
    │     작업 입력 테이블.
    │     작업명, 설명, 선행 작업, 소요 기간을 입력하는 행들과
    │     항목 추가 / 예시 불러오기 / 초기화 버튼을 포함한다.
    │
    ├── 📄 PredecessorSelect.tsx
    │     선행 작업 다중 선택 드롭다운.
    │     체크박스 목록으로 선행 작업을 선택하고 태그로 표시한다.
    │     테이블 overflow 문제를 해결하기 위해 Portal로 렌더링된다.
    │
    ├── 📄 CpmNode.tsx
    │     CPM 노드 하나의 렌더링 담당.
    │     원형 모드(작업명 + 소요 기간)와
    │     상세 모드(ES/DR/EF / 작업명 / LS/TF/LF 6칸 박스)
    │     두 가지 형태로 표시된다.
    │
    ├── 📄 NetworkGraph.tsx
    │     ReactFlow 기반 CPM 네트워크 그래프.
    │     useCpmLayout 훅으로 노드/엣지 상태를 받아 렌더링하고,
    │     원형↔상세 모드 전환, 노드 클릭 확장, 도움말 패널 등을 관리한다.
    │
    └── 📄 CpmGuide.tsx
          "CPM이란?" 가이드 페이지.
          CPM 개념 설명, 노드 구조, 계산 절차, 예시, 활용법을
          SVG 다이어그램과 함께 설명하는 전체 화면 안내 페이지다.


================================================================
각 폴더 요약
================================================================

  components/   화면에 직접 렌더링되는 UI 조각들
  hooks/        상태·로직을 재사용하기 위한 커스텀 React 훅
  lib/          UI 없이 독립적으로 동작하는 순수 함수·알고리즘
